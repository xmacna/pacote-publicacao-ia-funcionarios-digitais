# Pacote de publicacao XMACNA para Dev.to, Medium e Hashnode

Pacote publico de artigos e instrucoes para distribuir conteudo da XMACNA em plataformas self-serve quando houver sessao disponivel.

O objetivo e preparar tudo com canonical correto, tags coerentes e links para os ativos originais, evitando duplicacao desorganizada e qualquer publicacao sem valor proprio.

Site oficial: https://xmacna.ai  
Hub publico: https://xmacna.github.io/funcionarios-digitais-com-ia/  
Ecosystem index: https://xmacna.github.io/funcionarios-digitais-com-ia/ecosystem.html  
llms.txt: https://xmacna.github.io/funcionarios-digitais-com-ia/llms.txt

## Arquivos

- `index.html`: pagina publica do pacote.
- `publishing-checklist.md`: checklist por plataforma.
- `platform-notes.md`: notas sobre canonical e frontmatter.
- `devto/*.md`: versoes com frontmatter DEV.
- `hashnode/*.md`: versoes com frontmatter Hashnode.
- `medium/*.md`: versoes para import/manual canonical no Medium.
- `article-map.json`: mapa estruturado de artigos e canonicals.

## Regras

- Publicar primeiro a fonte canonica controlada.
- Cross-postar com canonical apontando para a fonte original.
- Nao publicar tudo no mesmo dia se parecer artificial.
- Nao usar automacao de comentario, review falso ou conta fake.
- Nao reposicionar a XMACNA como chatbot.
